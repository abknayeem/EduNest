EduNest - A Full-Stack Learning Management System

EduNest is a feature-rich, full-stack LMS built with the MERN stack. Developed as a final year university project, it provides a seamless, modern, and scalable e-learning experience for students, instructors, and administrators.

üöÄ Getting Started

To get a local copy up and running, please follow these steps.

Prerequisites

  • Node.js (v18 or later)
  • npm (or an alternative package manager)
  • MongoDB (a local instance or a cloud service like MongoDB Atlas)
  • Git

Installation from Repository

  1. 1. Clone the repository:

    git clone https://github.com/abknayeem/EduNest.git
cd EduNest

  2. 2. Setup the Backend:

    cd server
npm install

    Create a .env file in the /server directory and populate it with your credentials (use the provided .env file in the codebase as a template).

    Start the backend server:

    npm run dev

  3. 3. Setup the Frontend:

    Open a new terminal window and navigate back to the root `EduNest` folder.

    cd ../client
npm install

    Start the frontend development server:

    npm run dev

  4. 4. Access the Application:

    The application should now be running at http://localhost:5173.
